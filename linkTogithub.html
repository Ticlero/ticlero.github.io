# 준비물

VuePress 설치 Go To VuePress

Vuepress는 기본적으로 Node.js 8버전 이상을 요구합니다.

Node.js를 설치시 npm이 자동으로 설치됩니다.

yarn설치

(npm install --global yarn)

OS Windows에서 설치시 cmd로 진행하시면 됩니다.

# 설치

자신이 원하는 경로에 프로젝트 폴더를 생성하여 cmd(shell) 상에서 진행하시면 됩니다.

# install globally
yarn global add vuepress # OR npm install -g vuepress

# install workspace
yarn add vuepress -D

설치 후 실행을 위한 핵심파일만 받아집니다.

VuePress에서 권장하는 디렉토리 구조를 따라 진행해 주시면 됩니다.

구조참고

# 설정

  1. 프로젝트 폴더 안에 docs 폴더를 하나 생성 후 README.md 파일을 아래와 같이 만들어 줍니다.
README.md

# Hello VuePress
  1. 다시 상위 폴더로 올라가서 package.json파일을 열어서 script부분을 수정합니다.
package.json

...
"scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  },
...
  1. 현재 폴더(package.json이 존재하는)에서 deploy.sh 파일을 아래와 같이 생성합니다.
deploy.sh

set -e

npm run docs:build

# navigate into the build output directory
cd docs/.vuepress/dist

# if you are deploying to a custom domain
# echo 'www.example.com' > CNAME

git init
git add -A
git commit -m 'deploy'

git push -f git@github.com:(username)/(username).github.io.git master
  1. shell 상에서 sh deploy.sh를 실행합니다.

github page 연동시 public key 문제로 push가 안될 수 도 있습니다. 이 경우에는 여기를 눌러서 참고 하시기 바랍니다.

# 실행

자신의 프로젝트를 실행하는 방법은 간단합니다.

shell에서 cd 명령어를 통해 프로젝트 폴더로 이동 후

'yarn docs:dev' 를 하시게 되면 'localhost:8080'으로 접속하시면 됩니다.
